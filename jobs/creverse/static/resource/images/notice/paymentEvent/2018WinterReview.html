2018년 겨울학기 EVENT

생생한 Best 당첨 후기

청담, April어학원의 재학생이라면 누구나
EVENT의 주인공이 될 수 있습니다!

  • 다이슨 헤어드라이어 이*린 학생 학부모님 띠링띠링~♪ 이벤트 안내 문자가 왔고, 저희 아이에게 깜짝 선물로 줄 수 있는 기회가 주어진다면 너무나 기뻐할 것 같아 응모하게 되었습니다.
    그로부터 어느 날 학원으로부터 전화가 왔는데, 겨울학기 이벤트 당첨 소식을 알리는 것이었습니다.
    너무 기쁜 나머지 저도 모르게 나오려 하는 환호성을 겨우 참아냈답니다. 앞으로 경품을 사용할 때마다 기쁜 추억이 떠오를 것 같습니다^^
  • 다이슨 헤어드라이어 한*윤 학생 학부모님 이벤트 보고 다이슨 헤어 드라이기가 되면 좋겠다~ 생각했는데...
    어느 날 남편생일 축하노래를 열심히 부르고 있었는데 문자 알림 소리에 휴대폰을 확인 하는 순간 악!!! 기쁨과 놀람에 소리를 막 질렸어요~
    이벤트 선물도 너무 좋아요~!     원장님, 선생님들께도 감사 드립니다.
  • 커피머신 심*준 학생 학부모님 저에게도 이런 행운이 있네요^^ 청담을 오랫동안 다녔는데 이런 이벤트는 처음 참여하네요.
    아이가 청담어학원을 너무 재미있어하고 실력도 쑥쑥 늘어나서 계속 믿고 쭉 다닐 예정이라 이벤트 공지를 보자마자 바로 신청했는데 당첨까지 되었어요! 마침 갖고 싶던 제품이라 더 기분이 좋았네요. 감사합니다~~
  • 커피머신 조*연 학생 학부모님 겨울학기 3개월 선불 결제 이벤트라기에 매달 결제 타이밍 놓쳐 얼른 결제했을 뿐인데 생각도 못하게 이벤트 당첨되었다며 연락 주셨어요.
    그래서 유튜브 영상 검색까지 하고 기쁨 만끽하다 머신 와서 다시 한번 야호~~ 캡슐형으로 내려먹는 커피에 힘이 됩니다. 감사해요~~♥
  • 커피머신 전*찬 학생 학부모님 원하는 최고의 경품에 당첨되는 행운을 주셔서 너무 기쁘고 감사합니다.
    커피나 핫초코가 생각날 때마다 직접 내려 먹을 수 있어서 맛있고 편리하답니다. 이런 행복한 이벤트를 해주신 청담에게 깊은 감사 드립니다.
