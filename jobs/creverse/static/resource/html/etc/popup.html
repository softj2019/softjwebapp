C Mileage 소급 적용 안내

청담러닝 통합 멤버십, C Mileage 런칭과 함께
C Mileage가 아래의 기준으로 소급 적용되었음을 안내 드립니다.
적용 일자 :
6/1(월)
적용 기준 :
등록된 자녀가 5/31(일)까지 청담러닝 소속 브랜드를 재원한 기간
통합고객지원센터 운영 시간
재원기간 등급 C Mileage
185일 미만 C +0M
660일 미만 P +50M
1,000일 미만 Sp +250M
1,400일 미만 Ps +500M
1,400일 이상 V +800M
확인 방법 :
C Mileage 등급 및 보유 현황은 청담 올림(Allim) 앱 로그인 후 [C Mileage] > [나의 C Mileage]에서 확인 가능합니다.
C Mileage 자세히 보기
  • 단, 6개월 이상 퇴원 후 재입학한 학생은 재입학 시점부터 재원기간이 산정되었습니다.
  • 자녀가 2명 이상일 경우, 각 자녀에게 소급 적용된 C Mileage의 합산 Mileage로 적용됩니다.
  • 6/1(월)부터 C Mileage 적립 기준에 따라 C Mileage가 적립됩니다.
