공인인증서 발급 안내

  • 30만원 이상 신용카드 결제 시
    신용카드 공인인증서 절차가 의무화 되었습니다.
    (2005년 11월 1일 시행)
  • 공인인증서는 정부의 공인인증기관이 발행한 거래 인감 증명서입니다.
  • 전자거래 시 개인정보 뿐만 아니라 신원확인 등에 대한 법적인 효력도 받을 수 있습니다.

신용카드 공인인증서 발급 안내

  1. 1. 은행/보험/증권사/범용 공인인증서를 보유하고 있는 경우기존에 보유하고 계신 은행/증권사/보험용 공인인증서를 이용하시면 됩니다.
  2. 2. 은행/보험/증권사/범용 공인인증서를 보유하고 있지 않는 경우보유한 공인인증서가 없는 경우 먼저 거래하는 은행/보험/증권사에 가셔서
    해당 공인인증서를 받으셔야 합니다.
  3. 3. 거래하는 은행/보험/증권사가 없는 경우범용인증서를 이용하시면 한 번에 발급 받을 수 있습니다.
    (범용인증서는 한국정보인증 홈페이지에서 발급, 수수료=4,400원/년)
범용인증서 발급 받기

입금방법안내

가상계좌*를 통한 수강료 납부 방법

계좌번호 : 학생 본인의 수강료 납부 통지서에 기재된 가상 계좌번호 사용

  1. ① 전국의 모든 은행창구/인터넷뱅킹/폰뱅킹/ATM을 이용하여 무통장 입금 가능
  2. ② 청담닷컴 홈페이지 (http://www.chungdahm.com)에 학부모로 로그인 후, 마이페이지 → 수강료 납입 및 조회에서 계좌번호 및 금액 조회 가능
  3. ③ 수강료 납부 통지서에 기재된 금액과 다를 경우 입금 불가
  4. ④ 실시간 입금 확인 가능
  5. ⑤ 입금 가능한 시간 : 오전 7시 30분 ~ 오후 11시 30분
  6. ⑥ 개인별 가상계좌는 아이가르텐 수강료 및 출결카드비 납부 이외의 다른 용도로는 사용할 수 없음
  7. ⑦ 개인별 가상계좌는 다른 사람과 공유하여 사용할 수 없음
  8. ⑧ 학부모들의 수수료 부담을 줄이기 위해 타 은행과는 현재 개발중입니다.

*학생 개개인에게 고유의 가상계좌번호를 부여하고, 그 계좌로 수강료를 입금하는 방식

온라인 카드 결제 방법

  1. 1. 납입하고자 하는 클래스를 선택 후, 온라인 카드 결제 항목의 [결제하기]를 클릭해주세요. 교실수업 수강료는 필수적으로 결제하셔야 하며, 나머지 항목 중 결제를 희망하는 항목에 Check한 후
    “결제하기” 버튼을 누르세요. (중복선택 가능합니다.)
  2. 2. 결제금액이 30만원 이상일 경우, 결제 시 공인 인증서가 필요합니다.
    공인인증서 발급을 원치 않으시면 check 박스를 이용하여 분할 결제도 가능합니다.     범용 인증서 발급 받기 버튼을 누르면, 금융결재원 전자 인증센터로 이동합니다.
  3. 3. 거래용 정보 입력 화면에서 성명/금액/E-mail/휴대폰을 확인 후,
    [결제하기] 버튼을 클릭해주세요.
  4. 4. 신용카드 승인창이 뜨면, 결제방식, 결제내용, 금액, 할부기간 확인 후 결제를 진행해 주세요.
  5. 5. 신용카드 결과를 확인하세요.
